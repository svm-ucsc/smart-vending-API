**items**
----
  Returns json data about all items.

* **URL**

  /items

* **Method:**

  `GET`
  
*  **URL Params**

  None

* **Data Params**

  None

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `[{},{},]`

* **Sample Call:**

  ```javascript
    fetch(BASE_URL + "/items", {mode:'no-cors'});
  ```
